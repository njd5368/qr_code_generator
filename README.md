# qr_code_generator
A command line interface for generating QR codes wirtten in Rust. 

```bash
$ qr_gen [OPITONS] TEXT
```

```bash
$ qr_gen [OPTIONS] FILE
```

Options:

`-o, --output TEXT` The file for the output.

`-s, --style TEXT` The style for the qr code.

`-i, --image FILE` An image to put in the center of the qr code.
